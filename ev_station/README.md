# EV Charging Station Management System

A web-based application developed using Django that allows users to book EV charging slots and track charging status. Admin users can manage stations, slots, and charging operations.

## Features
- User authentication and login
- EV charging slot booking
- Charging status tracking
- Admin dashboard for management

## Tech Stack
- Python
- Django
- SQLite (MySQL supported)
- HTML, CSS
